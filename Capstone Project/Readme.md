Real Estate Website – Capstone Project (HTML & CSS).

A simple and fully responsive real estate website built using HTML5 and CSS3 as part of the Web Development Capstone project.

Technologies Used-
•HTML5 for structure
•CSS3 for layout and styling
•Flexbox & Grid for responsiveness
•Google Fonts (Montserrat)

Project Features-
•Responsive layout (desktop + mobile)
•Hero banner 
•Search / Filter bar
•Featured Properties section
•Trusted Agents section
•Why Choose Us section
•Contact / Enquiry form
•Stats section (1200+ properties)

How to Run-
•Download the project folder
•Open index.html in any browser
•No additional setup needed
